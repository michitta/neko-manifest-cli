# neko-manifest-cli

## 🍵 Usage

```
neko-manifest-cli.exe <server_name> <loader> <loader_version> <mc_version>
```

## 🪅 Example of usage

```
neko-manifest-cli.exe earth fabric 0.15.7 1.20.1
```

## ⚙️ Supported loaders

| Loader    | Manifest CLI | Launcher |
| ------    | :-----: | :-----: |
| Fabric    |   ✅   |   ✅   |
| Forge     |   ❌   |   ✅   |
| Vanilla   |   ❌   |   ✅   |
| Other     |   ❌   |   🟡   |
